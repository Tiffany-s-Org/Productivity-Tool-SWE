--Running the program--

step 1:
Inside the IDE terminal -
nmp install express
npm install hbs
npm install mongoose
npm install nodemon 

step 2:
Inside the IDE terminal -
nodemon src/index.js

This will run the program.

step 3:
go to a web browser and type "localhost:3000"
